# Decision Log Template

```md
## YYYY-MM-DD HH:mm:ss +08:00 - US-XXX - Short Title

- Status: chosen | blocked | changed | fallback
- Decision: What the agent chose.
- Reason: Why this is the best option under repository constraints.
- Alternatives Considered: Options not chosen and why.
- Files Changed: Paths touched by this task.
- Validation: Commands run and result.
- Follow-up: Remaining work, if any.
```
