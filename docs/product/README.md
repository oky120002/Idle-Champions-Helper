# product 文档入口

本目录存放项目级边界、路线图和跨模块长期规则；只在需要产品范围或全局约束时加载。

## 先读哪份

- `docs/product/idle-champions-roadmap.md`：项目定位、阶段目标、范围边界与明确不做的内容
- `docs/product/documentation-governance.md`：文档职责、精简原则、按需加载和更新触发条件
- `docs/product/mobile-compatibility-guidelines.md`：移动端布局约束，尤其是横向滚动禁令

## 适用场景

- 需要确认当前阶段做什么、不做什么
- 需要判断新规则应该写进 `AGENTS.md`、`README.md` 还是专题文档
- 需要确认移动端交互是否越界
