# investigations 文档入口

- 作用：收纳排查、验证、环境确认和历史问题记录；只在需要复现或核对证据时进入。

## 主题入口

- `docs/investigations/runtime/README.md`：本地运行、预览、浏览器行为与页面异常
- `docs/investigations/repository/README.md`：仓库结构、Git 与历史配置问题
- `docs/troubleshooting-log.md`：已沉淀成摘要的通用问题

## 读取建议

- 先看 `docs/troubleshooting-log.md` 有没有现成摘要
- 需要复现过程、边界或证据时，再进入对应主题 `README.md` 和单篇排查文档
