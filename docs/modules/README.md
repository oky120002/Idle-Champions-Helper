# modules 文档入口

本目录存放模块设计稿与模块级补充资料；开发前优先加载目标模块目录，不需要通读全部设计稿。

## 模块索引

- `docs/modules/champions/`：英雄筛选、英雄详情、立绘页设计
- `docs/modules/formation/`：阵型编辑设计
- `docs/modules/pets/`：宠物图鉴设计
- `docs/modules/presets/`：方案存档设计
- `docs/modules/shared-components/`：共享组件治理与目录
- `docs/modules/user-data/`：个人数据导入设计

## 使用约定

- 新模块先建独立目录，再落设计稿
- 设计稿里区分“当前已实现”与“后续计划”
- 若一个模块下文档数量继续增长，优先在该模块目录增加局部 `README.md`
