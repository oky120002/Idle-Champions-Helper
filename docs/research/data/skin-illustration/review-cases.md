## 2026-04-16 delta 审计追加经验

- `112`：候选 `sequence 0 / frame 28` 确实更收、更连贯，但 current 也完整可读，当前保守不写 override。
- `351`：delta 分数很高，但主要是大剑从第二连通域变成更靠近主体；肉眼差异很小，不写 override。
- `298`：候选让漂浮饮料更靠近主体，但它依旧明显是主题道具，不写 override。
- `496`：存在轻微改善，但收益不强，不建议升级为 override。
- `282`、`210`、`103`、`59`、`306`：current-vs-candidate 对比后都没有实质改善，可直接沉淀为“主题本意 / 候选无收益”的参考样本。

新增可复用规则：current-only 的 detached 审计只适合生成候选池；真正决定是否继续人工，优先看 delta 审计；delta `promising` 只代表“值得深看”，不代表“可以直接写 override”。

## 后续追加建议

以后新增人工结论时，继续按“新增样本 / 新增可复用规则 / 被推翻或修正的旧经验”三段追加，这样这份经验库才能持续累积，而不是每轮重写一遍。
