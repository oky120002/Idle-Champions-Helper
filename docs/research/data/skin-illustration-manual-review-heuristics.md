## 当前稳定下来的规则

### 什么情况下值得写 override

当前真正落地为 override 的样本，基本都满足至少一条：

- 替代 pose 明显修复“主体过小 / 画布过空”，如 `133` 星运伊芙琳
- 替代 pose 明显收回默认 frame 中横向外展过大的武器或肢体，主体更集中，如 `473` 飞升阿斯代伦
- 替代 pose 把明显偏窄或异常展开的构图拉回同英雄组更自然的范围，如 `368` 半精灵变异哨兵、`520` 舞厅蔚

核心标准不是“有点不一样”，而是“肉眼有明确且稳定的优势，而且更像正式主立绘”。

### 什么情况下不值得写 override

以下类型当前都应保守处理：

- 候选只有轻微尺寸变化，没有更自然，如 `362`、`290`、`515`、`111`、`452`、`550`
- 候选虽然更紧凑，但引入更严重的连接错误，如 `123` 星运弗里，手部与法杖断开，最终已撤回 override
- detached 道具 / companion / 浮空组件在多个候选 pose 中持续存在，如 `282`、`298`、`306`、`351`、`496`
- 主题本来就是小体型 / 玩偶 / 毛绒 / 宝宝，如 `470`、`471`、`409`
- 主题本来就天然更宽、更高或更复杂，如 `367` 半人马、`210` 玻璃罩、多主体包装类主题

## 当前最有用的判断经验

- detached 不等于错。先问它是不是主题元素、在多个候选里是否都持续存在；如果是，更可能是主题构图本意，而不是 frame 选错。
- 更紧凑不等于更对。连接关系与肢体完整性永远比尺寸分数更重要；`123` 是典型反例。
- 同主题组持续出现的 detached 元素，优先判成主题本意。样例：`282` 漂浮眼 / 书、`298` 漂浮饮料、`306` 酒壶 + 蝙蝠、`496` 月牙 / 星星 / companion。
- 真正的错误 frame 通常会有更强对照信号：明显修复断手 / 断杖 / 断武器、显著减少莫名外展、把主体拉回画布中心、且相邻 frame 也持续更优。

## 推荐复核流程

1. 先看候选是怎么被筛出来的：尺寸审计更看主体占比；alpha 审计更看 detached 元素是否属于主题本意。
2. 始终先看 current，再看 top 候选 8 张，确认是否存在“压倒性更对”的 pose。
3. 不确定时检查相邻 frame；如果只是单帧碰巧更紧凑，通常不值得写 override。
4. 最终裁决优先级：
   - 肢体 / 武器 / 手部连接是否完整
   - 主体是否清楚、稳定、像正式主立绘
   - detached 元素是否属于主题本意
   - 尺寸、fill、面积是否更接近组内中位数

不要把第 4 条放到前面。

## 当前经验带来的直接结果

截至 2026-04-16，这些经验已经支撑两个动作：

- 第一轮尺寸审计候选池基本跑空
- 第一轮 alpha 审计候选池在补完 top 样本人工复核后也已跑空

所以下一步若还要系统扩样，不能只重复“看宽高面积偏差”或“看 detached 连通域数量”，而应继续做 current-vs-candidate delta 审计。

## 2026-04-16 第 1 轮 delta 审计补充经验

- `112`：候选 `sequence 0 / frame 28` 确实更收、更连贯，但 current 也完整可读，当前保守不写 override；若继续深挖，它仍是第一优先级回看样本。
- `351`：delta 分数很高，但主要是大剑从第二连通域变成更靠近主体；肉眼差异很小，不写 override。
- `298`：候选让漂浮饮料更靠近主体，但它依旧明显是主题道具，不写 override。
- `496`：存在轻微改善，但收益不强，不建议升级为 override。
- `282`、`210`、`103`、`59`、`306`：current-vs-candidate 对比后都没有实质改善，可直接沉淀为“主题本意 / 候选无收益”的参考样本。
- 新增可复用规则：
  - current-only 的 detached 审计只适合生成候选池；真正决定是否继续人工，优先看 delta 审计
  - delta `promising` 只代表“值得深看”，不代表“可以直接写 override”
  - delta 为负或几乎为 0 的样本，通常可以直接归档为主题本意
  - 长武器 / 长尾道具一旦刚好与主体接上，alpha 连通域分数会突然变好很多，但肉眼观感可能只变一点；`351` 是典型假阳性
  - 若 top 候选只是一串相邻 frame 轻微摇摆、没有带来新的主体阅读感，通常更像动画相位差，不值得仓促写 override
