# 国内免费静态托管平台调研

- 调研日期：2026-04-12
- 背景：Gitee Pages 已下线，寻找国内免费静态网站托管方案

---

## 1. 国内平台现状

### 1.1 腾讯云 CloudBase 静态托管

| 项目 | 详情 |
|------|------|
| 费用 | 新用户免费体验环境，每月 3000 点资源额度 |
| 存储额度 | 1 GB |
| 流量额度 | 5 GB/月 |
| 自定义域名 | ✅ 支持（需 ICP 备案） |
| HTTPS | ✅ |
| SPA 支持 | ✅ |
| CI/CD | 支持 CLI 部署（`tcb hosting deploy`） |
| 备案要求 | **自定义域名必须备案**；使用默认域名（cloud.tencent.com 子域）不需备案 |
| 超量价格 | 0.21 元/GB 流量 |

**评价**：最靠谱的国内免费方案，但免费额度有限（5GB 流量/月）。对于个人游戏辅助站可能够用。需要腾讯云账号和实名认证。

### 1.2 阿里云 OSS 静态网站托管

| 项目 | 详情 |
|------|------|
| 费用 | 无永久免费额度，按量计费 |
| 存储费用 | ~0.12 元/GB/月 |
| 流量费用 | ~0.50 元/GB（CDN 可降到 0.20 元/GB） |
| 自定义域名 | ✅ 支持（需 ICP 备案） |
| HTTPS | ✅ |
| SPA 支持 | ✅（需配置回退 404 → index.html） |

**评价**：没有真正的免费额度，个人小站月费约 1-5 元。不是"免费"方案。

### 1.3 又拍云

| 项目 | 详情 |
|------|------|
| 费用 | 有「又拍云联盟」免费 CDN 计划（需网站底部加又拍云 logo） |
| 性质 | CDN 服务，不是完整静态托管 |
| 备案要求 | 需要备案 |

**评价**：只是 CDN 加速层，不是托管方案。可作为 GitHub Pages 的加速补充。

### 1.4 七牛云

| 项目 | 详情 |
|------|------|
| 费用 | 每月 10 GB 存储 + 10 GB 流量的免费额度（需实名） |
| 性质 | 对象存储，可配置为静态网站托管 |
| 自定义域名 | ✅（需备案） |
| HTTPS | ✅（免费证书） |

**评价**：有免费额度，但配置较复杂，且本质是对象存储而非静态托管。需要备案。

## 3. ICP 备案问题

### 什么是 ICP 备案

中国大陆要求：使用中国大陆服务器的网站必须在工信部完成 ICP 备案。

### 什么时候需要备案

| 场景 | 是否需要备案 |
|------|------------|
| 使用国内云服务（腾讯云/阿里云/七牛等）+ 自定义域名 | **必须备案** |
| 使用国内云服务的默认子域名 | 通常不需要 |
| 使用海外服务（GitHub Pages/Cloudflare/Vercel） | **不需要** |

### 备案的麻烦

- 需要域名 + 服务器（云服务商可提供）+ 真实身份认证
- 审核周期 1-3 周
- 每年需要年审
- 个人备案有一定限制（不能有交互功能等）

### 不备案的选择

- 使用 GitHub Pages 默认域名（username.github.io）→ 免备案
- 使用 Cloudflare Pages 默认域名 → 免备案
- 接受国内访问稍慢的现实

## 4. 推荐方案

### 方案 A：GitHub Pages（推荐 ✅）

**最简单、最省心的方案。**

- 零成本，零配置
- 不需要备案
- 国内访问速度可以接受（非实时应用，游戏辅助站对延迟不敏感）
- 如果后续觉得慢，可以加 Cloudflare CDN 加速

### 方案 B：Cloudflare Pages

**GitHub Pages 的优质替代。**

- 同样免费，零备案
- 国内访问通常比 GitHub Pages 直连快
- 自动集成 GitHub 仓库
- 构建配置更灵活

### 方案 C：双线部署（GitHub Pages + 腾讯云 CloudBase）

**如果确实需要国内极速访问。**

- GitHub Pages 作为海外线路
- 腾讯云 CloudBase 作为国内线路
- DNS 按地域分流（需付费 DNS 服务）
- 复杂度高，**第一阶段不建议**

### 最终建议

> **先用 GitHub Pages。** 如果玩家反馈访问慢，再迁移到 Cloudflare Pages 或加 CDN。不要在部署上花太多时间，先把产品做出来。

| 阶段 | 方案 |
|------|------|
| MVP / 第一版 | GitHub Pages（默认域名） |
| 如果访问慢 | Cloudflare Pages（免费 + 更好的全球 CDN） |
| 如果需要国内极速 | GitHub Pages + Cloudflare CDN，或腾讯云 CloudBase |
