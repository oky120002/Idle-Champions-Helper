# troubleshooting 主题入口

- 作用：收纳流程、环境、认证、网络、部署、运行等非单点代码问题；默认先看问题摘要，再只开对应单篇。

## 先读哪篇

- GitHub Pages 首次部署失败：`docs/troubleshooting/pages-first-deploy.md`
- 受限 Codex 会话内 Playwright 无法正常启动：`docs/troubleshooting/playwright-sandbox.md`
- 文档索引重复与环境路径耦合导致文档老化：`docs/troubleshooting/documentation-aging.md`
- 当前环境直连 GitHub 可能卡住：`docs/troubleshooting/github-connectivity.md`
- 旧工作树残留预览服务污染当前浏览器回归：`docs/troubleshooting/stale-preview-port.md`
